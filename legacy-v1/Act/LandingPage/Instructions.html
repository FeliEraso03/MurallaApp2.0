All questions to use MurallaGRAPH like a PRO

Check if you are in Basic mode and click on Show Grid.

The grid should now be visible on the map and a button to deactivate it should appear.

Check if you are in Basic mode and click on Show Heat Map.

The heat map should now appear on the map, and a control panel will be visible, along with a button to hide the heat map.

Through the control panel, you can start and stop the heat map's movement, as well as move forward or backward in time.

Check if you are in Basic mode and click on Routes.

You can now create and delete routes by selecting two points on the map, as well as view points of interest around those routes.

You can also change the route's destination by selecting one of the two points again and view the information about the points of interest.

Check if you are in Advanced mode and click on Graph.

You should now see a panel that allows you to activate graph creation, edit them, display the graph's direction, or clear them.

With graph creation activated, you should be able to create nodes by selecting points on the map.

By selecting two nodes on the map with graph creation activated, you should be able to create an edge between these two nodes.

With the directed graph activated, you should be able to see the direction of the edges in the graph.

With Edit graph activated, you should be able to edit the nodes and edges.

Check if you are in Advanced mode and click on Algorithms, it is necessary to have a graph already created.

You will see a panel with the algorithms, and we select the algorithm.

By selecting two nodes from the graph and clicking the Start button, it will show us the algorithm solution.

Check if you are in Advanced mode and click on Layers.

You should see a panel with the different layers and a button to add more layers

By clicking the button, you can enable or disable a layer, adjust its opacity using the slider, and move the layer's position up or down with the left slider control.
