# Regla de Diseño UI
ESTRICTAMENTE PROHIBIDO el uso de Emojis dentro de la interfaz de usuario en todo el proyecto.
En su lugar, SIEMPRE usar íconos SVG profesionales como los provistos por la librería `lucide-react`, imágenes, CSS puro o equivalentes modernos que se adapten al sistema de diseño oscuro y naranja empleado. Reemplaza emojis en texto explicativo, botones y selectores con SVGs.
